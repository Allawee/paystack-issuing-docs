---
title: Cards
description: Issue, manage, fund, and control physical and virtual cards.
---

Cards are the core of Allawee's issuing product. Create virtual or physical cards linked to a card program, set spending controls, and manage the card lifecycle.

---

## Create Card

```http
POST /cards
```

### Request Body

| Field | Type | Required | Description |
|---|---|---|---|
| `program` | string | ✓ | Card program ID |
| `customer` | string | ✓ | Customer ID |
| `amount` | number | | Initial funding amount (in minor units) |
| `reference` | string | | Your unique reference |
| `cardHolderName` | string | | Name printed on card |
| `debitSource` | string | | Account to debit for initial funding |
| `fundingSource` | string | | Source for ongoing card funding |
| `debitCurrency` | string | | Currency to debit in |
| `fxHash` | string | | FX quote hash for cross-currency funding |
| `controls` | object | | Spending controls |
| `metadata` | object | | Custom key-value data |

### Spending Controls

| Field | Type | Description |
|---|---|---|
| `allowedMcc` | array | Allowed merchant category codes |
| `blockedMcc` | array | Blocked merchant category codes |
| `allowedCategories` | array | Allowed merchant categories |
| `blockedCategories` | array | Blocked merchant categories |
| `allowedMerchants` | array | Allowed merchant IDs |
| `blockedMerchants` | array | Blocked merchant IDs |
| `spendingLimits` | array | Spending limit rules |

### Example Request (Virtual NGN)

```json
{
  "program": "cpg_xxxxx",
  "customer": "cus_xxxxx",
  "amount": 500000,
  "reference": "card-ref-001",
  "cardHolderName": "JOHN DOE",
  "fundingSource": "act_xxxxx",
  "controls": {
    "spendingLimits": [
      { "amount": 10000000, "interval": "daily" }
    ]
  }
}
```

### Example Request (Virtual USD)

```json
{
  "program": "cpg_xxxxx",
  "customer": "cus_xxxxx",
  "amount": 5000,
  "debitSource": "act_xxxxx",
  "debitCurrency": "NGN",
  "fxHash": "fxq_xxxxx",
  "fundingSource": "act_xxxxx"
}
```

## List Cards

```http
GET /cards
```

Standard [pagination, filtering, and sorting](/getting-started/#pagination) parameters apply.

---

## Get Card

```http
GET /cards/:id
```

Returns a single card by ID.

---

## Get Card Secrets

:::caution
This endpoint uses the **secure base URL**: `https://secure.allawee.com`
:::

```http
GET /cards/:id/secrets
```

Returns sensitive card data including PAN, CVV, and expiry date.

---

## Get Card Balance

```http
GET /cards/:id/balance
```

Returns the current card balance.

---

## Fund Card

```http
POST /cards/:id/fund
```

Add funds to a card.

| Field | Type | Required | Description |
|---|---|---|---|
| `amount` | number | ✓ | Amount to add (in minor units) |
| `debitSource` | string | | Account to debit |
| `debitCurrency` | string | | Currency to debit in |
| `fxHash` | string | | FX quote hash for cross-currency |

---

## Withdraw from Card

```http
POST /cards/:id/withdraw
```

Withdraw funds from a card back to the funding source.

---

## Get Funding Source

```http
GET /cards/:id/funding-source
```

Returns the funding source configured for this card.

### Get Funding Source Balance

```http
GET /cards/:id/funding-source/balance
```

## Reserve Card

```http
POST /cards/:id/reserve
```

Reserve a physical card from inventory for assignment.

---

## Link Card by ID

```http
POST /cards/:id/link
```

Link an unassigned card to a customer.

| Field | Type | Description |
|---|---|---|
| `reference` | string | Your unique reference |
| `customer` | string | Customer ID |
| `cardHolderName` | string | Cardholder name |
| `fundingSource` | string | Funding source account |
| `controls` | object | Spending controls |
| `metadata` | object | Custom data |

---

## Link Card by PAN

:::caution
This endpoint uses the **secure base URL**: `https://secure.allawee.com`
:::

```http
POST /cards/link
```

Link a card using the full PAN instead of the card ID.

## Update Card

```http
PUT /cards/:id
```

Update card spending controls and metadata.

| Field | Type | Description |
|---|---|---|
| `controls` | object | New spending controls (overwrites existing) |
| `metadata` | object | New metadata (overwrites existing) |

This endpoint uses the **secure base URL**: `https://secure.allawee.com`
:::

| Field | Type | Required | Description |

## Reset Card PIN

This endpoint uses the **secure base URL**: `https://secure.allawee.com`

## Block Card

```http
POST /cards/:id/block
```

Temporarily block a card. The card can be unblocked later.

---

## Unblock Card

```http
POST /cards/:id/unblock
```

Unblock a previously blocked card.

| Field | Type | Description |

## Terminate Card

```http
POST /cards/:id/terminate
```

Permanently terminate a card. This action cannot be undone.

Alternative endpoint:

```http
DELETE /cards/:id
```
