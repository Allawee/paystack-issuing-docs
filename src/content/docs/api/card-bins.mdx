---
title: Card Bins
description: Manage card BIN ranges for your card programs.
---

Card BINs (Bank Identification Numbers) are the first 6–8 digits of a card number that identify the issuer and program. Manage your card BIN allocations through these endpoints.

---

## List Card Bins

```http
GET /card-bins
```

Standard [pagination](/getting-started/#pagination) parameters apply.

---

## Get Card Bin

```http
GET /card-bins/:id
```

Returns a single card BIN by ID.

---

## Create Card Bin

```http
POST /card-bins
```

Create a new card BIN allocation.

---

## Update Card Bin

```http
PUT /card-bins/:id
```

Update an existing card BIN.

---

## Request Approval

```http
POST /card-bins/:id/request-approval
```

Submit a card BIN for approval.

---

## Get Card Bin Account Logs

```http
GET /card-bins/:id/account-logs
```

Returns activity logs for a card BIN.

### Download as CSV

```http
GET /card-bins/:id/account-logs/csv
```
