---
title: Payment Authorizations
description: Create and manage standing payment authorizations.
---

Payment authorizations allow you to set up pre-approved payment mandates that can be executed without per-transaction approval.

---

## Create Payment Authorization

```http
POST /payment-authorizations
```

Create a new payment authorization.

---

## List Payment Authorizations

```http
GET /payment-authorizations
```

Standard [pagination, filtering, and sorting](/getting-started/#pagination) parameters apply.

---

## Get Payment Authorization

```http
GET /payment-authorizations/:id
```

Returns a single payment authorization by ID.

---

## Revoke Payment Authorization

```http
POST /payment-authorizations/:id/revoke
```

Revoke an active payment authorization. Revoked authorizations cannot be used for future payments.
