---
title: Accounts
description: Create and manage bank accounts for your business and customers.
---

Accounts represent bank accounts used for receiving and holding funds. Allawee supports three account types:

- **Main** — Your primary business operating account
- **Sub** — Separate accounts for organizing funds
- **Virtual** — Dedicated accounts for receiving inward payments, linked to a settlement account

---

## Create Account

```http
POST /accounts
```

### Request Body

| Field | Type | Required | Description |
|---|---|---|---|
| `name` | string | ✓ | Account name |
| `currency` | string | ✓ | `NGN` or `USD` |
| `type` | string | ✓ | `main`, `sub`, or `virtual` |
| `customer` | string | | Customer identifier |
| `settlementAccount` | string | | Settlement account (virtual accounts only) |
| `reference` | string | | Your unique reference |
| `depositChannels` | array | | Deposit methods (currently `bank-account`) |
| `metadata` | object | | Custom key-value data |

### Example Request

```json
{
  "name": "Customer Collections",
  "currency": "NGN",
  "type": "virtual",
  "settlementAccount": "act_xxxxx",
  "customer": "cus_xxxxx",
  "depositChannels": ["bank-account"],
  "reference": "my-ref-001"
}
```

### Example Response

```json
{
  "id": "act_xxxxx",
  "name": "Customer Collections",
  "currency": "NGN",
  "type": "virtual",
  "status": "active",
  "depositChannels": [
    {
      "type": "bank-account",
      "accountNumber": "1234567890",
      "bankName": "Wema Bank",
      "bankCode": "035"
    }
  ],
  "createdAt": "2025-01-15T10:30:00.000Z"
}
```

## List Accounts

```http
GET /accounts
```

Standard [pagination, filtering, and sorting](/getting-started/#pagination) parameters apply.

---

## Get Account

```http
GET /accounts/:id
```

Returns a single account by ID.

---

## Get Account Balance

```http
GET /accounts/:id/balance
```

Returns the current balance for the account.

---

## Get Balance History

```http
GET /accounts/:id/balance/history
```

Returns the balance ledger history for an account. Supports pagination.

### Download as CSV

```http
GET /accounts/:id/balance/history/csv
```

---

## Download Statement

```http
GET /accounts/:id/statement/pdf
```

Downloads a PDF statement for the account.

---

## Add Deposit Channel

```http
POST /accounts/:id/deposit-channels
```

Add a new deposit channel (e.g., bank account) to an existing account.

---

## Update Deposit Channel Account Name

```http
PUT /accounts/:id/deposit-channels/update-account-name
```

Update the account name on a deposit channel.
