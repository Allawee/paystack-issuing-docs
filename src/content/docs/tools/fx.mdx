---
title: FX Rates & Quotes
description: Get foreign exchange rates and quotes.
---

Get real-time FX rates and lock in quotes for cross-currency card funding and payments.

---

## Get Current FX Rate

```http
GET /tools/fx/rate
```

### Query Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `from` | string | ✓ | Source currency (e.g., `NGN`) |
| `to` | string | ✓ | Destination currency (e.g., `USD`) |

---

## Get FX Quote

```http
POST /tools/fx/quote
```

Get a locked FX quote for a specific amount.

| Field | Type | Required | Description |
|---|---|---|---|
| `from` | string | ✓ | Source currency |
| `to` | string | ✓ | Destination currency |
| `amount` | number | ✓ | Amount to convert |

---

## Get Buy Quote

```http
POST /tools/fx/quote/buy
```

Get a quote for buying a specific amount of foreign currency.

| Field | Type | Required | Description |
|---|---|---|---|
| `fromCurrency` | string | ✓ | Currency to pay with |
| `toCurrency` | string | ✓ | Currency to buy |
| `toAmount` | number | ✓ | Amount of target currency to buy |

### Example Request

```json
{
  "fromCurrency": "NGN",
  "toCurrency": "USD",
  "toAmount": 10000
}
```

---

## Get Sell Quote

```http
POST /tools/fx/quote/sell
```

Get a quote for selling a specific amount of currency.

| Field | Type | Required | Description |
|---|---|---|---|
| `fromCurrency` | string | ✓ | Currency to sell |
| `toCurrency` | string | ✓ | Currency to receive |
| `fromAmount` | number | ✓ | Amount to sell |
