---
title: Merchants
description: Look up merchants and merchant categories.
---

Utility endpoints for working with merchant data, useful for card spending controls.

---

## List Merchants

```http
GET /tools/merchants
```

Returns all merchants in the Allawee merchant directory.

---

## Get Merchant Categories

```http
GET /tools/merchants/categories
```

Returns all merchant category codes (MCCs) with their descriptions. Use these when configuring card spending controls.

---

## Get Merchant

```http
GET /tools/merchants/:id
```

Returns a single merchant by ID.
