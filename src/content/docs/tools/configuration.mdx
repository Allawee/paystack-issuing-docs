title: Configuration
description: View and update your account configuration.
---

Manage your account-level configuration including default payment settings.

---

## Get Configuration

```http
GET /tools/configuration
```

Returns your current account configuration.

---

## Update Payment Configuration

```http
PUT /tools/configuration/payment
```

### Request Body

| Field | Type | Description |
|---|---|---|
| `payInAccount` | string | Default account for receiving payments |
| `payOutAccount` | string | Default account for sending payments |
| `fees` | object | Default fee configuration |
| `fees.bankTransfer` | string | Fee ID for bank transfers |
| `fees.billPayment` | string | Fee ID for bill payments |

### Example Request

```json
{
  "payInAccount": "act_xxxxx",
  "payOutAccount": "act_xxxxx",
  "fees": {
    "bankTransfer": "fee_xxxxx",
    "billPayment": "fee_xxxxx"
  }
}
```
