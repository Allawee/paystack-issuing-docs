The Allawee dashboard provides a visual interface for managing your financial infrastructure.

## Key Capabilities

- **Issue cards** — Create and manage physical and virtual cards
- **Assign accounts** — Create bank accounts for customers
- **Make transfers** — Send payments directly from the dashboard
- **Pay bills** — Process bill payments
- **Monitor activity** — Track transactions, events, and API usage

---

## Team Management

Add team members through **Settings > Team**. Available roles:

| Role | Description |
|---|---|
| **Admin** | Full access to all dashboard features |
| **Developer** | Access to API keys and technical settings |

---

## Bank Accounts

Create bank accounts directly from the dashboard:

1. Navigate to **Accounts**
2. Click **Create Account**
3. Choose account type: **Main**, **Sub**, or **Virtual**
4. Configure account settings and deposit channels

---

## Card Issuing

Issue cards from the dashboard:

1. Navigate to **Cards**
2. Create or select a **Card Program**
3. Configure card settings (network, currency, spending controls)
4. Issue cards to customers

Card programs created through the dashboard follow the same approval workflow as the API — physical programs require approval while virtual programs go live immediately.
